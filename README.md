# OpenAI Sample (Node.js)

Пример проекта для работы с OpenAI API (Responses API).

## Установка

```bash
npm install
```

## Настройка

Скопируйте `.env.example` в `.env` и укажите свой API-ключ:

```bash
cp .env.example .env
```

Ключ можно получить на [platform.openai.com](https://platform.openai.com/api-keys).

## Запуск

```bash
npm start
```

Ответ выводится в консоль.
